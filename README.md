This was a challenge to make something interesting with vanilla JavaScript in less than 1kb.

It plays [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) using the microphone
as a seed via as an oscilloscope.

[Code golfing](https://en.wikipedia.org/wiki/Code_golf) it down smaller was my own challenge.

You can see it at [https://samlev.dev/gol-oscilloscope.html](https://samlev.dev/gol-oscilloscope.html)

Currently at 826 bytes.
